# Modal project
En este projecto se crea una ventana modal en la cual los usuarios pueden ver la paleta de colores (predefinida) del fondo de pantalla

## lineas usadas:

  - getElementById()
  - querySelector()
  - addEventListener()
  - forEach()
  - textContent
  - createElement()
  - appendChild()
  - createTextNode()
  - classList.add
  - classList.remove
  - style.backgroundColor
  - style.color
